# iTerm2 Workspace

Opens a 6-pane development layout:

```
| lvim (66% x 66%) | claude1 |
| lvim             | claude2 |
| fish1  | fish2   | fish3   |
```

## Usage

```bash
~/.me/iterm/workspace /path/to/project
```

## Prerequisites

Grant Accessibility permission to iTerm2:

```
System Settings > Privacy & Security > Accessibility > iTerm (enable)
```

Required for pane resizing via keyboard shortcuts.
